1) Il problema e la funzione da minimizzare

Il problema considerato riguarda il posizionamento di una struttura di approvvigionamento in un punto \(f(\chi , \nu)\) situato all’interno di una mappa (con \(\chi , \nu\) incognite relative alla posizione della struttura), in maniera tale per cui tutti la somma di tutte le distanze dei clienti che intendono approvvigionarsi da essa (note all’interno del dataset e indicate con \(p_i = (x_i,y_i)\)) sia minore possibile.
I clienti vengono rappresentati nello spazio bidimensionale attraverso il seguente grafico:

Fornite le coordinate del cliente i-esimo, la distanza tra la struttura ed il cliente viene così calcolata:
\(d(f,p_i) = log((\chi - x_i)^2+1)+log((\nu - y_i)^2+1)\)
La funzione da minimizzare restituisce la posizione \(f(\chi, \nu)\) calcolata come:
\(f(\chi,\nu) = \sum^{n}_{i=1} d(f,p_i)\)

Si assume che i costi da sostenere per la costruzione della struttura siano fissi (non dipendenti dalla posizione considerata) e che le coordinate \(\chi,\nu\) della struttura non siano scelte all’interno di una serie di punti noti a nostra disposizione (ma siano calcolati dall’algoritmo).

Funzione di distanza

L’implementazione della funzione della distanza in codice R è la seguente:

fun = function(chi, nu, df){
  res = 0
  for (i in c(1:nrow(df))) {
    res = (log((chi - df$x[i])^2 + 1)) + (log((nu - df$y[i])^2 + 1)) + res
  }
  return(res)
}

2) Gradiente della funzione di distanza

Il gradiente della funzione da minimizzare è definito dalla seguente formula matematica:

\[\nabla f(n_k) = \begin{cases} \frac{2(\chi - x_i)}{((\chi - x_i)^2+1)}\\ \frac{2((\nu - y_i))}{((\nu - y_i)^2+1)} \end{cases} \]

L’equivalente della funzione in codice R è definito dal seguente script:

gradient = function(chi, nu, df){
  res1 = 0
  res2 = 0
  for(i in c(1:nrow(df))){
    res1 = (2*(chi-df$x[i]))/((chi-df$x[i])^2 + 1) + res1
    res2 = (2*(nu-df$y[i]))/((nu-df$y[i])^2 + 1) + res2
  }
  return(c(res1, res2))
}

3) Implementazione dell’algoritmo Steepest Descent Method e risoluzione del problema con esso

Il metodo scelto per l’ottimizzazione della funzione è il metodo del gradiente (Steepest Descent Method), ossia un metodo che permette di ottimizzare rapidamente una funzione tramite un semplice algoritmo; questo algoritmo è in grado di restituire un punto che rappresenta un ottimo locale (di cui però non si conosce l’ottimalità globale), cioè un punto in cui la funzione assume un valore ottimo (minimo o massimo) in un intorno considerato.
Il metodo del gradiente possiede le seguenti componenti fondamentali:
  • un punto di partenza \(x_k\)
  • un passo \(\alpha_k\) che stabilisce di quanto la funzione si sposta all’interno della mappa ad ogni iterazione
  • una direzione \(d_k\) lungo la quale muoversi, massimizzando con il segno + e minimizzando con il segno -
  • una funzione da ottimizzare \(f\)
  • il gradiente della funzione \(\nabla f\)
Prevede l’esecuzione dei seguenti passi:
  1. Si sceglie un punto iniziale \(x_k\) fornito all’algoritmo o scelto casualmente
  2. Si calcola il gradiente della funzione \(\nabla f(x_k)\):
  3. Si associa alla direzione \(d_k\) il gradiente della funzione con il segno + o - a seconda del tipo di ottimizzazione prestabilita: \(d_k =\pm\nabla f(x_k)\)
  4. Si calcola il punto successivo della funzione (fornita una dimensione del passo \(\alpha_k\)) :
    \(x_{k+1} = x_k + \alpha_k*d_k\)
  5. Si effettua lo stop dell’algoritmo sotto tre condizioni:
    • Il gradiente nel punto \(x_{k+1}\) è nullo
    • Viene raggiunto un valore di precisione inferiore ad una soglia predeterminata \(e\), valutato con \(| f(x_{k+1})- f(x_k)| < e\) o in alternativa \(||\nabla f(x_{k+1})|| < e\)
    • Si raggiunge un numero massimo di iterazioni prestabilite

NB Fornito un punto (\(\chi, \nu\)) generico in input all’algoritmo che si occuperà della ottimizzazione si otterrà in output una coppia di coordinate (\(\chi_{opt}, \nu_{opt}\)) che rappresenterà l’ottimo locale in cui la funzione è minimizzata

Algoritmo Steepest Descent Method

Il procedimento che è stato descritto nel paragrafo precedente si traduce in codice R con il seguente script:

gradientDescent = function(df, v.init, grad, passo, iters){
  par = loss = matrix(NA, nrow = iters + 1, ncol = 2)
  par[1,] = v.init
  for(k in 1:iters){
    loss[k,] = grad(par[k,1], par[k,2], df)
    par[k+1,] = par[k,] - passo * loss[k,]
  }
  return(list(par = par))
}

Restituirà una lista contenente le soluzioni, di cui l’ultimo elemento rappresenta l’ottimo, il punto in cui l’algoritmo converge e termina l’esecuzione.
NB Si sceglie di mantenere il valore del passo pari a 0.01 per tutte le sue esecuzioni.

5) Implementazione dell’algoritmo Stochastic Gradient Descent e risoluzione del problema con esso

Si procede ad implementare il metodo Stochastic Gradient Descent, una versione dell’algoritmo del gradiente che risulta estremamente utile in presenza di grandi dataset su cui deve essere eseguita una ottimizzazione di funzioni; il problema sottoposto è caratterizzato dalla presenza di solo 100 osservazioni nel dataset, in queste condizioni può sfruttare le proprie peculiarità per restituire velocemente i punti di ottimo in cui la funzione viene massimizzata (in quanto possiede una velocità di esecuzione superiore al Deepest Descent Method).
L’algoritmo prevede un campionamento del dataset che permette di valutare tramite mini-batch il gradiente della funzione in maniera approssimata ad un numero limitato di punti considerati: il calcolo della funzione-distanza ad ogni iterazione non verrà efettuato per 100 clienti ma verrà approssimato in funzione di 15 clienti.
Questo procedimento permette di ridurre enormemente i tempi di esecuzione e di restituire un punto ottimo senza che esso risulti troppo influenzato dalle osservazioni selezionate.
Il codice R che implementa l’algoritmo è il seguente:

stoch_gradDescent = function(df, start, gradient, lr, iters){

  beta = start
  
  betaIter = data.frame(matrix(NA, nrow = iters, ncol = 2))
  
  set.seed(42)
    
  # now iterate using mini batches of randomly sampled  data, updating theta each step
  for (i in 1:iters) {
    # randomly sample 5 items from the combined xy data frame
    df_temp = data.frame(as.matrix( df[sample(nrow(df), 15), ] ))
   
    beta = beta - lr  * gradient(beta[1], beta[2], df_temp) 
      
    # save the beta values for iteration i to a matrix for future plotting
    betaIter[i,] = beta
      
 } # end for loop
 return(betaIter)
}
